# InfraGmen（インフラGメン）
### リアルタイム通信監視・判定支援ツール

## 🚀 概要
本ツールは、OSが提供する公開情報（psutil）を用いて、**現在確立されているネットワーク通信をリアルタイムに一覧表示し、人間による判断を支援するための可視化ツール**です。

バックグラウンド通信を構造化して表示することで、開発環境や運用現場において「未知の接続」や「不要なポートの利用」を迅速に察知し、セルフデバッグやセキュリティ確認を行うことを目的としています。

## ✨ 特徴
* **リアルタイム・プロセス特定**: 通信を行っているプロセス名（実行ファイル名）を瞬時に特定して表示。
* **二段階フィルタリングシステム**:
    * **ホワイトリスト**: 信頼できる既知の通信を「通常ログ」として整理。
    * **ブラックリスト**: 警戒すべき通信を「警告ログ（強調表示）」として分離。
    * **未登録HTTPS判定**: 443番ポート通信の中で未定義のものをハイライトし、新規ドメインの確認を容易にします。
* **表示レイアウトの最適化**: 日本語文字が含まれる環境でも各カラムが垂直に整列するよう、動的なパディング処理を実装。
* **高パフォーマンス設計**: DNSキャッシュ機能を搭載し、システム負荷を最小限に抑えつつ高速なホスト名解決を実現。

## 🛡️ 設計思想
本ツールは **「自動遮断・自動判定」を行わない、純粋な可視化ツール** です。

* **非干渉**: 既存の通信環境やパケットの内容には一切干渉しません。
* **判断支援**: あくまで「人間が状況を把握し、判断するための材料を揃える」ことに特化しています。

IDS/IPSの代替ではなく、**運用・保守・開発現場における「状況把握の迅速化」**にフォーカスしています。

## 🛠 技術スタック
* **Language**: Python 3.11+
* **Libraries**: 
    * `psutil` (システム情報取得)
    * `tkinter` (GUI)
    * `threading` (非同期並列処理)
    * `unicodedata` (文字幅計算)

## ⚙️ 設定方法
`whitelist.csv` および `blacklist.csv` を作成し、監視ルールを定義します。

| pattern | type | comment |
| :--- | :--- | :--- |
| 8.8.8.8 | ip | Google DNS |
| 445 | port | Microsoft-DS (Alert) |
| example.com | keyword | 任意のドメインキーワード |

## ⚠️ 免責事項・注意事項
* 本ツールは学習および個人のネットワーク管理を目的として作成されています。
* 管理者権限がない環境では、一部のプロセス名が取得できない場合があります。
* 業務利用時は、必ず所属組織のセキュリティポリシーに従ってください。

## ✍️ 作成背景
開発現場において「通信が多すぎて、現状把握に時間がかかる」という課題を解決するために作成しました。
情報を整理し、管理者が自信を持って環境の状態を把握できる環境を提供することを目指しています。
